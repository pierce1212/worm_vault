# ISO 26262 part6软件部分的clause7.2指出，软件架构内容应包含静态设计和动态设计，静态设计方面，要描述组件所实现的功能以及组件之间的交互接口，包括数据接口和函数接口；动态方面，需要描述功能的处理顺序（数据流）  以及时序（调度以及先后顺序），数据流以功能为单位，描述一个功能从输入、到中间的处理、再到输出，完整的数据流向；时序同样是以功能为单位，描述不同组件之间的调度以及先后顺序关系
