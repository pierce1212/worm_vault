# OpenAI Codex 快速入门

## 什么是 OpenAI Codex？
OpenAI Codex 就像是一个**代码翻译官**。你用平常说话的方式（比如中文或英文）告诉它你想做什么，它就能直接帮你写出计算机代码。

*   **核心身份**：它是著名的自动编程助手 **GitHub Copilot** 背后的“大脑”。
*   **主要技能**：它精通几十种编程语言（如 Python, JavaScript, HTML, PHP 等）。

## 它能做什么？
1.  **自动写代码**：你描述需求，它直接生成代码。
2.  **代码补全**：你写了一半，它能猜出你后面想写什么并帮你填好。
3.  **解释代码**：如果你看不懂某段复杂的代码，它可以像老师一样逐行解释给你听。

## 怎么用？（小白示例）
你不需要记忆复杂的编程语法，只需要在代码编辑器里输入一句简单的指令：

**场景：我想在网页上做一个按钮**
*   **你的指令**：`创建一个红色的网页按钮，点击后弹出“你好”`
*   **Codex 自动生成的代码**：
    ```html
    <button style="background-color: red; color: white;" onclick="alert('你好')">点我</button>
    ```

## 核心优势
*   **零门槛**：用自然语言（人话）就能编程。
*   **高效率**：帮你省去了查阅手册和手动输入重复代码的时间。
