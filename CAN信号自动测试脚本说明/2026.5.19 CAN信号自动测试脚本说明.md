# 测试版支持TXRX、NODE选择，DBC选择

指令
```
powershell -File E:\canspace\autotest\ETAS_CAN_Rx_AutoTest.ps1 -SelectDbc -SelectNode -SelectMode  交互选择

只想看检测项目、不真正测试
powershell -ExecutionPolicy Bypass -File E:\canspace\autotest\ETAS_CAN_BiDir_AutoTest.ps1 -DbcPath E:\canspace\autotest\E0XProjectCASU.dbc -Mode RX -NodeName ETAS -ListSignals


单测 RX 的 CIC_P_1_CRC
.\ETAS_CAN_BiDir_AutoTest.ps1 -TestDirection RX -OnlySignal CIC_P_1_CRC
如果你还要手动选择 DBC / 节点
.\ETAS_CAN_BiDir_AutoTest.ps1 -SelectDbc -SelectNode -TestDirection RX -OnlySignal CIC_P_1_CRC
如果脚本在 E:\canspace\autotest
cd E:\canspace\autotest
.\ETAS_CAN_BiDir_AutoTest.ps1 -SelectDbc -SelectNode -TestDirection RX -OnlySignal CIC_P_1_CRC
```



```
1. 一定要注意，你的代码工程需要调整，变量不能被篡改否则误报错。
2. 代码要与板子的工程保持一致，否则误报。
3. DBC要与脚本放在同一个文件夹下
```
